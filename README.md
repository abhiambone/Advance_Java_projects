# 🚀 ServletProject_4 – Java Servlet CRUD Real Web Application 🌐
---------------------------------------------------------------------

> [!IMPORTANT]
> ⚠️ **Before Running the Project**
> 🚀 You do **not** need to add the MySQL Connector/JAR file manually.
> 📦 The MySQL dependency is already configured in the **pom.xml** file.
> ✅ Simply import the project as a Maven project and build it.

## 📖 About the Project
✨ **ServletProject_4** is a dynamic Java Web Application developed using **Java Servlets**, **JDBC**, **HTML**, and **MySQL Database**. This project demonstrates how web applications interact with databases to perform user management operations such as registration, login, viewing records, and updating user information.
💡 It is an excellent project for learning the fundamentals of **Java Enterprise Development**, **Database Connectivity**, and **CRUD Operations** using Servlets.

## 🏗️ Project Structure

### 📂 WEB-INF
- 📄 `web.xml`
  - Servlet configuration and deployment settings.

### 📂 DatabaseQueries
- 🗃️ `TableCreationQuries`
  - Contains SQL scripts for database table creation.

### 📂 com.sub.nit (Servlet Package)

#### 🔑 LoginServlet.java
Handles user login and authentication.

#### 📝 RegisterServlet.java
Processes new user registration requests.

#### 👀 ViewServlet.java
Retrieves and displays user records from the database.

#### ✏️ UpdateServe.java
Accepts update requests from users.

#### 🔄 UpdateProcessServlet.java
Processes and saves updated user information.
